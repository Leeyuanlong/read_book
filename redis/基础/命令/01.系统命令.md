# 系统命令

### -keys pattern
    *号为通配符
    
### -ttl key
    查看key的生命周期
   
### -expire key seconds
    设置key的过期时间

### -persist key
    移除key的过期时间
    
### -type key
    查看key的类型

### -select 0 
    切换数据库
### -exists key  
    判断key是否存在
    
### -DUMP key
    序列化给定的key，并返回序列化的值
    
### -move key db
    移动key到库
    
### -random key 
    随机返回一个元素
    
### -rename key newkey
    修改key的名称
